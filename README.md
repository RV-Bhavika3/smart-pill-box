# smart-pill-box
IoT-based Smart Pill Box with real-time medicine reminders using NodeMCU, Arduino Nano, OLED display, and RTC module.

## Project Description
The Smart Pill Box is an IoT-based project that reminds users to take their medicine on time using an RTC module, OLED display, LED, and buzzer.

## Features
- Displays time using RTC (DS3231)
- Medicine reminder alerts
- OLED display for notifications
- LED and buzzer alert system

## Components Used
- NodeMCU ESP8266
- Arduino Nano
- DS3231 RTC Module
- OLED Display (0.96")
- LED
- Buzzer

## Technologies Used
- Arduino IDE
- Embedded C / Arduino Programming

## How It Works
The RTC module keeps track of real-time. When the preset medicine time matches the current time, the system triggers the buzzer and LED while displaying a reminder on the OLED screen.
